Manage and display Rotten Tomatoes Movies.

== Description ==

Get Movies from RT Server.


- Widget included
- All Movies from Rotten Tomatoes
- a lot of movie details
- actors, genres, etc


= ShortCode =

For Movies:

	[nt_movies]

For DVDs

	[nt_movies type=dvds]

== Installation ==

1. Upload plugin folder to the /wp-content/plugins/ directory
2. Activate the plugin through the Plugins menu in WordPress
3. Now you are done. Enjoy the plugin!!!
